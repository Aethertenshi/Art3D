# Art3D Engine Core (C++)

A lightweight C++20 game engine viewport built using SDL3 and Dear ImGui (Docking branch).

## Features
- **C++20 Modules:** Optimized build times and modern architecture using standard `.ixx` modules.
- **Dear ImGui (Docking):** Full Dockspace layout with hierarchical Scene Explorer and Properties Inspector.
- **SDL3 Renderer Integration:** Cross-platform windowing, event polling, and viewport rendering.
- **FPS Limiter:** Frame rate locked at 120 FPS to optimize CPU consumption.

## Setup & Build
1. Open the project folder in VS Code.
2. Make sure you have MSVC installed (Visual Studio 2022/2025/2026).
3. Trigger the build task `Build Art3DC++ (SDL3)` using `Ctrl + Shift + B`.
4. Run the executable in `x64/Debug/Art3DC++.exe`.
